# Detect Open Ports on Sub-Domains


### Use Naabu:-

```bash
naabu -list subs.txt -p - -o naabu-full-portscan.txt
```


### Run httpx to detect the Web Services:-

```bash
httpx -l naabu-full-portscan.txt -o httpx-naabu-full-portscan.txt
```
